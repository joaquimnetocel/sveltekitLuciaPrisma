# SVELTEKIT PRISMA AUTH.JS EXAMPLE

## LUCIA PACKAGES

```bash
npm install -D lucia
npm install @lucia-auth/adapter-prisma
npm install arctic
```

## DEVELOPMENT

```bash
npx prisma db push
npx prisma generate
npm run dev
```

## HOW TO CREATE THE CREDENTIALS

- [FOR GITHUB](https://docs.github.com/en/apps/oauth-apps/building-oauth-apps/creating-an-oauth-app)
